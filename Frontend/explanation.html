How this explanation works

PhishDefender uses a machine learning model to classify emails based on patterns found in the message content. The explanation shown here highlights common warning signs and reassuring signs found in the email text, sender details, and wording.

These explanations are designed to help users understand the decision, but they are not a guarantee that an email is definitely safe or definitely malicious. They should be used as decision support alongside normal caution.

How to read the results

  • Overall explanation: quick human readable reason
  • LIME: important words for this specific email
  • SHAP: strongest model feature contributions
  • Confidence bars: how strongly the model leaned each way
